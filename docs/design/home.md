## Hero セクション

### レイアウト
- スライドショー形式（画像 3枚）
- 画像なし（テキスト＋CTA のみ）ではなく、フルワイド背景画像

### コピー
```
AI・ITを活用したい。でも、誰に頼めばいい？
集客・採用・業務効率化・システム開発まで、
あなたのビジネスを一緒に作り上げるITパートナー。
```

### CTA ボタン
- 「お問い合わせはこちら」→ `/contact`
- 「サービスを見る」→ `/service`

### 画像
- スライド 3枚（PC 用 + SP 用 = 計 6 枚）
- PC: 1920×1080px / SP: 750×1334px
- テーマ: ①ビジネス・パートナーシップ ②AI・テクノロジー活用 ③成果・成長
- **Issue #23 にて素材準備中（ユーザー対応）**

## お悩みセクション

### 見出し
```
こんなお悩みありませんか？
```

### レイアウト
- 3カラム（カテゴリ別グループ型）
- 各カラム：イラスト（上）＋ カテゴリ名 ＋ 悩みリスト（箇条書き）
- 最下部に締めの一言

### カラム構成

| カラム | イラスト | 悩み |
|--------|---------|------|
| AI・DX活用 | AI・データ操作系（unDraw: `artificial intelligence` / `data`） | ・ChatGPTを触ったことはあるが、自社業務への活かし方がわからない<br>・自動化したい作業はあるが、どのツールを選べばいいか判断できない |
| Webマーケ | 分析・グラフ系（unDraw: `analytics` / `online marketing`） | ・広告費をかけているのに問い合わせが来ない・費用対効果が見えない<br>・SNSを更新しているが集客につながっている実感がない |
| Web・システム開発 | 開発・コードレビュー系（unDraw: `programming` / `code review`） | ・制作会社に見積を取ったら数百万円と言われて断念した<br>・作ったWebサイトが古くなっているが更新できていない<br>・エンジニアに依頼しても話が噛み合わず、思った通りに仕上がらない |

### 締めの一言
```
そんな課題を解決するため、伴走型で支援します。
```

### 画像素材
- unDraw（無料・商用可・SVG）を使用
- テーマカラーはサイトのプライマリカラー（`oklch(0.25 0.08 250)`）に合わせる

## Service セクション

### 役割
ホーム上の導線。詳細は `/service` に委ねる。

### レイアウト
- 横3カラム
- 各カラム：サービス名 ＋ ベネフィット文 ＋ CTA（「詳しく見る」→ `/service`）

### サービス構成

**AI・DX活用支援**
```
毎日の繰り返し作業をAIで自動化。
書類作成・データ集計・返信対応を仕組み化し、
社員がコア業務に集中できる環境をつくります。
```

**Webマーケティング**（公式LINE構築・運用を含む）
```
数値を見ながら改善し続ける、ROIが見えるマーケ支援。
集客からリピートまで一貫して支援します。
```

**ホームページ制作・アプリ開発**
```
要件を整理し、適正コストでスピーディに開発。
事業視点で一緒に設計・開発するパートナーとして伴走します。
```

### CTA
- 「詳しく見る」→ `/service`
